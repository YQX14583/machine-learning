## 1. 问题定义与理解

本项目使用 CIFAR-10 数据集完成图像分类任务。该数据集包含 10 个类别、60,000 张 `32×32` 的彩色图像，类别分别为：`airplane`、`automobile`、`bird`、`cat`、`deer`、`dog`、`frog`、`horse`、`ship`、`truck`。

本次作业需要完成：

1. 图像数据读取与预处理。
2. 图像分类模型设计。
3. 模型训练、测试与结果评估。
4. 对实验结果进行分析与总结。

本项目最终只保留一个经过优化的 `CNN` 模型用于提交。

## 2. 数据分析与处理

### 2.1 数据集特点

- 总样本数：60,000
- 训练集：50,000
- 测试集：10,000
- 每类样本数：6,000
- 图像大小：`32×32×3`

由于图像尺寸较小，细节有限，部分类别之间存在较强相似性，例如猫和狗、鹿和马，因此模型需要具备较强的空间特征提取能力。

### 2.2 数据预处理

本项目采用如下预处理方式：

1. 图像转为张量。
2. 使用 CIFAR-10 常用均值和标准差进行标准化。
3. 对训练集进行随机裁剪和随机水平翻转。
4. 使用 `AutoAugment(CIFAR10)` 进一步增强训练样本多样性。

标准化参数如下：

- Mean: `(0.4914, 0.4822, 0.4465)`
- Std: `(0.2023, 0.1994, 0.2010)`

### 2.3 数据划分

训练集进一步按 `9:1` 划分为训练集和验证集，并采用分层划分方式，以尽量保证各类别分布均衡。

## 3. 模型构建

本项目最终采用一个优化版卷积神经网络 `CNN` 作为唯一分类模型。

### 3.1 选择 CNN 的原因

相比简单的全连接网络，CNN 更适合图像任务，原因如下：

1. 卷积层可以提取局部纹理、边缘和形状特征。
2. 参数共享有助于降低模型复杂度。
3. 池化操作可以增强平移不变性。
4. CNN 能保留图像空间结构信息，因此在 CIFAR-10 上更有效。

### 3.2 优化版 CNN 结构

为了尽可能提升单模型效果，本项目对 CNN 做了强化设计，主要包括：

1. 使用卷积 `stem` 提取底层特征。
2. 使用多个残差卷积块提升特征表达能力。
3. 在残差块中加入 `SEBlock` 通道注意力，增强有效特征响应。
4. 激活函数采用 `GELU`，使训练过程更平滑。
5. 在分类头前使用全局平均池化，减少参数量并提升泛化能力。

整体来看，该模型兼顾了卷积网络的局部特征提取能力、残差结构的稳定性和注意力机制的特征筛选能力。

### 4.2 超参数设置

本项目使用如下主要超参数：

- Batch Size：128
- Epoch：30
- 随机种子：42
- 优化器：AdamW
- 学习率调度：Cosine Annealing
- 正则化：Dropout、Weight Decay、Label Smoothing
- 训练技巧：Gradient Clipping、Early Stopping

### 4.3 训练策略说明

为了提升单模型性能，本项目采用了以下优化策略：

1. 使用更强的数据增强方法。
2. 使用 `AdamW` 结合权重衰减提升泛化能力。
3. 使用标签平滑降低模型对训练标签的过度自信。
4. 使用梯度裁剪提升训练稳定性。
5. 使用早停策略防止过拟合。
